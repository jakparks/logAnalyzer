# logAnalyzer

Before Running:
Ensure that you have node.js installed on your system

To Run (From admin directory):

node server.js

Will listen on port 3000
Home URL: http://localhost:3000/logAnalyzer

Dependencies:
- express
- body-parser
- method-override
- morgan
- http
- path
- properties-reader
- passport/passport-local
- mongodb
- bluebird
- scp
- log4js
- zlib
- stream
